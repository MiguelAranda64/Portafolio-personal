Proyectos

Mantefix

Proyecto de optimización de mantenimiento correctivo y preventivo para equipos de maquinaria en industrias y planificación de tareas críticas. Las actividades a realizar dependian del rol del usuario logeado, si era tecnico podía completar algunas tareas asignadas. Mientras tanto el gerente administraba todas las actividades hechas y por hacer de los tecnicos al mando. El supervisor puede agregar maquinas nuevas al area y asignarlas con ID, ubicación, planta y otros datos. La app móvil se programó en React Native, se utilizó Django Rest Framework para backend.

Mantefix en Movil

Recursos humanos

Proyecto que simula ser un sistema de gestión de recursos humanos. Facilita la administración de toda la información relacionada con los empleados, desde el reclutamiento y la contratación hasta la administración de pre nóminas, beneficios, la capacitación, evaluación del desempeño y asistencias. La plataforma tambien permite el inicio de sesión con diferentes roles. La aplicación web fue hecha con React y Django Rest Framework.
